Projeto para testar conhecimentos básicos

Este projeto tem como objetivo principal demonstrar as habilidades em criação de um CRUD utitilizando o Spring Framework.

Linguagens utilizadas


JAVA 11

Spring framewrok
O Spring Framework é um framework de aplicação e container de inversão de controle para a plataforma Java

Thymeleaf
O Thymeleaf é uma template engine para projetos Java que facilita a criação de páginas HTML.

HTML 5

Bootstrap 5.0

Banco de dados relacional (h2)
H2 é um sistema de gerenciamento de banco de dados relacional escrito em Java. Ele pode ser incorporado em aplicativos Java ou executado no modo cliente-servidor.
